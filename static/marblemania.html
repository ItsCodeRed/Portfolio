Marble Mania
When I was only 12 years old, I started learning how to make games. At that time, I was using GameMaker Studio, and I was avoiding learning programming because I thought it would be too hard. But at some point along the process, I decided to try out the Unity game engine.

Of course, with this engine, you HAD to learn programming to get anywhere, so I started following tutorials, and started work on my first Unity game! Inspired off of my YouTube videos, I decided to make the game a marble racing, platformer game. After many monthes of work, I evantually released Marble Mania on October 1st, 2017.

Since I already had a pretty large following on my YouTube channel, a suprising number of people played the game! At the time of writing, the game has over 15,500 downloads on GameJolt! Which is just insane.

Over the years, I've updated the game quite a few times, including a huge graphics overhaul on the one year anniversary of the game. But sometime during 2020, I lost interest in the project, and that's where the story ends.
